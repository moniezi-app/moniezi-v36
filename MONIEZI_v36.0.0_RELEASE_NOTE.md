# MONIEZI Pro Finance v36.0.0 — Simple Continuation

This package is the next development version of MONIEZI Pro Finance v35.0.3.

## What changed

- Displayed application version changed to **36.0.0**.
- Main package and GitHub release markers changed to v36.
- Service-worker cache identifier changed so the new code refreshes correctly.
- The reviewed Cloudflare owner-key implementation is retained.

## What did not change

- Main MONIEZI IndexedDB database names.
- Receipt-image database.
- Local-storage keys and preferences.
- Device identity.
- License activation record and offline binding.
- PWA customer-facing name and established app identity.
- Backup format and application data model.

There is **no v35-to-v36 migration step**. Version 36 continues the same application and uses the same existing browser data.
