# MONIEZI v35.0.0 — Pro Finance Fundraising-Ready Baseline

This package is based on the verified MONIEZI v32.2.2 Pro Finance build and promoted to the v35 line.

## Purpose

v35.0.0 keeps the normal MONIEZI Pro Finance app as the owner/admin business app while retaining the internal fundraising-tracking tools that were added in the v32 equity line.

## Included owner-side fundraising tools

- Equity module
- Stakeholders
- Guided share issuance
- Cap table summary
- SAFE / convertible instrument tracker
- Investment reservations
- Investor package drafting and preview
- CSV exports for equity/reservation records

## Product role

MONIEZI Pro Finance is the internal owner/admin app. It is used to run the business and track fundraising records internally.

MONIEZI Raise remains the separate investor-facing companion product for public/private signup pages, six-step investor onboarding, document download, and pledge/reservation submission workflows.

## Notes

This app records business and fundraising data for internal tracking. It does not replace attorney review, securities-law compliance, payment processing, board approvals, or a registered fundraising intermediary where one is legally required.
