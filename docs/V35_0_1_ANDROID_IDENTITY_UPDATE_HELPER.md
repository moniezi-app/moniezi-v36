# MONIEZI v35.0.1 — Android Identity + Update Helper

This build is based on MONIEZI Pro Finance v35.0.0.

## Changes

- PWA name set to **MONIEZI Pro Finance V35**.
- PWA short name set to **MONIEZI 35** for Android home-screen identification.
- Browser title set to **MONIEZI Pro Finance V35**.
- Manifest ID aligned to the GitHub Pages repo path: `/moniezi-pro-finance-v35/`.
- Service-worker cache version bumped so Android/Chrome sees a fresh build.
- Added **Settings → Update → App Update / Refresh Build**.

## New Settings update controls

- Check for Update
- Reload App
- Reset Demo Data Only
- Export Backup
- Import Backup

Use these controls before clearing Chrome site data. Clearing site data can erase local MONIEZI records stored on the device.
